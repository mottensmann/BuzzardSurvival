
## R-code for survival analysis using combined dead and live encounter data of buzzards *Buteo buteo*

  - **R\_code.html** contains the code for all analyses. In addition, we
    provide our data files in the folder called `data`, which should be
    placed in a subfolder of the working directory in order to repeat
    analyses.

## **`data`**

This folder contains raw data and results of our analysis. The raw data
files are listed below.

  - **nao.txt** contains North Atlantic Oscillation (NAO) indices
    derived from
    <https://climatedataguide.ucar.edu/climate-data/hurrell-north-atlantic-oscillation-nao-index-pc-based>

  - **ringing\_data.csv** contains individual specific data

  - **resighting\_data.csv** contains resightings of individuals

## **`R`**

  - Contains custom scripts that were used for some of the computations,
    e.g. bootstrapping procedures.
